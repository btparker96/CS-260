Image formats and their purpose

  • JPG: large color palettes without exorbitantly increasing file size but no transparent pixels
  • GIF: go-to for simple animations but limited color palette
  • PNG: Great for nayting that's not a photo or animated
  • SVG: vector-based graphics so great for anything needing scalability. Same use cases as PNGs
